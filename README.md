# Retail-sale-project
The `RetailSaleAnalyzer` class analyzes retail sales data by cleaning missing values, calculating total sales per product, identifying the best-selling product, and computing average daily sales. It also provides visualizations of sales trends over time and sales distribution per product, helping businesses make data-driven decisions

## Overview
`RetailSaleAnalyzer` is a Python class that helps analyze retail sales data. It can:
- Clean the dataset by removing missing values.
- Calculate total sales for each product.
- Identify the best-selling product.
- Calculate the average daily sales.
- Visualize sales trends over time.
- Visualize sales distribution per product.

## Requirements
To use the `RetailSaleAnalyzer` class, make sure you have the following Python libraries installed:
- `pandas` for data manipulation.
- `matplotlib` for data visualization.
